# VeilCore

Cryptographic proof of prior possession for cannabis genetics, built on Midnight Network.

A private record created when genetics arrive at a tissue culture lab. The intake record is hashed locally in the browser; only the hash would be committed. Zero raw genetic data leaves the facility.

## Status — read first
Working proof-of-concept, not production.
- Real: browser-side SHA-256 hashing, tamper detection, certificate flow, full UI
- Simulated: the Midnight commit. No Compact contract or testnet transaction yet.
- Next: Compact contract + live testnet integration.
